Assignment Checker
A web-based tool for detecting plagiarism and analyzing writing quality in academic assignments. Upload documents or paste text to receive similarity scores and quality assessments.

Key Features
Checks for copied content
Evaluates writing quality
Identifies problem areas
Responsive design for all devices

Usage
Clone this repository
Open index.html in your browser
Choose to either:

Upload a document (PDF/DOCX)
Paste your text directly
View your analysis results

Developer Notes
This frontend prototype demonstrates:
Modern CSS techniques (Grid, variables)
Accessibility best practices
Clean UI design patterns
